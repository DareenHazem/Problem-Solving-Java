# Level 0 - Problem Solving in Java

Welcome to my **Level 0 problem-solving sheets** in Java!  
This folder contains my beginner-level Java exercises organized by topic. 
Each sheet is a separate **package**, and each Java file inside is a solution to a problem.

## Structure

- **Sheet_1** : Data types and basic conditions  
- **Sheet_2** : Loops and iteration problems  
- **Sheet_3** : Arrays and basic array algorithms  
- **Sheet_4** : String manipulation problems  
- **Sheet_5** : Functions and simple algorithms  
- **Sheet_6** : Mathematical problems
